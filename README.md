# Operations Planning Dashboard

This project demonstrates data-driven operations planning using forecasting, inventory analysis, and process improvement concepts.

## Project Goals
- Forecast product demand
- Analyze inventory risk
- Calculate reorder points and safety stock
- Simulate operational change requests
- Present business insights clearly

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Excel

## Project Structure
- `data/` → input datasets
- `notebooks/` → analysis scripts and notebooks
- `output/` → charts, reports, and final files
