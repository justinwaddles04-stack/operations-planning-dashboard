# Operations Planning Dashboard

This project demonstrates data-driven operations planning using forecasting, inventory analysis, and structured change control concepts.

## Business Problem
Operations teams must balance demand variability, inventory levels, and process changes while minimizing stockouts and inefficiencies.

This project simulates how an operations or supply chain analyst would:
- Forecast demand
- Determine reorder points
- Identify inventory risks
- Track operational changes through a structured change control process

## Key Features
- Demand trend analysis across multiple products
- Inventory planning using safety stock and reorder point calculations
- Automated identification of reorder risks
- Change control log to simulate regulated operational changes
- Status tracking and reporting for change management

## Outputs
- Demand trend visualization (`demand_trends.png`)
- Inventory planning summary (`inventory_summary.csv`)
- Change control status summary (`change_status_summary.csv`)

## Tools Used
- Python (Pandas, NumPy, Matplotlib)
- Excel (data structuring and validation)

## What This Demonstrates
- Data-driven decision making
- Process improvement mindset
- Operational planning and analysis
- Understanding of structured change management concepts
